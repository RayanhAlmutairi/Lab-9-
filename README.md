# Lab-9-
This project aims to predict whether a borrower will fully repay a loan using the LendingClub dataset by applying two machine learning models: a Decision Tree and a Random Forest. The data was analyzed by exploring key features such as FICO scores, interest rates, and loan purposes, and categorical variables were converted into numerical form for modeling. A Random Forest model with 600 trees was trained and compared to a single Decision Tree using classification reports and confusion matrices. The results showed that the Random Forest achieved better performance with about 85% accuracy, although both models struggled with the minority class due to data imbalance.
